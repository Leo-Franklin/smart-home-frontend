## Choosing a Design for smart-home-frontend

Given this is a **Vue.js smart home dashboard** with cameras, devices, schedules, and settings:

**Top recommendations:**
1. **`linear`** — Clean dashboard aesthetic, sidebar nav, semantic status colors — best overall fit
2. **`vercel`** — Minimal, dark mode, technical — great for tech-forward smart home
3. **`stripe`** — Professional and trustworthy, clean data tables — good for settings-heavy views
4. **`cursor`** — Dark AI-native panels — excellent if the app has a dark/technical vibe
