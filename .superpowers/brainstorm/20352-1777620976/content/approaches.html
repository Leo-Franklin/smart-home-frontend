Cron 可视化选择器 — 3 种方案

选择一个你最喜欢的方向（可多选）

A

预设卡片 + 高级折叠

默认展示常用预设卡片（每天凌晨、工作日、每小时等），点击即选。底部有「自定义」展开原始 cron 输入框。

每天凌晨 2 点 工作日早 8 点 每小时 每周日 + 自定义…
▶ 已选：每天凌晨 2 点 → 0 2 * * *

优点

  • 最易上手
  • 无需学习 cron
  • 快速完成配置

缺点

  • 预设有限
  • 高级用户仍需手写
B

可视化时间构建器

分拆 cron 的每个字段：时间选择器 + 星期多选 + 重复频率下拉。实时生成 cron 并显示人类可读描述。

触发时间
02:00
重复
每天 ▾
星期（全选 = 每天）
一 二 三 四 五 六 日
✓ 每周一到周五 凌晨 2:00 → 0 2 * * 1-5

优点

  • 直观灵活
  • 覆盖常见+进阶场景
  • 实时预览

缺点

  • 无法覆盖极端 cron 语法
  • 开发量稍大
C

A + B 组合：分段切换

顶部 Tab 切换「快速选择」（预设卡片）和「自定义」（可视化构建器）两种模式。高级用户还能展开原始 cron 输入。

快速选择 自定义时间 高级 (Cron)
每天凌晨 2 点 工作日早 8 点 每 30 分钟 周末整天
切到「自定义时间」→ 时间构建器 / 切到「高级」→ 原始 cron 框

优点

  • 满足所有用户层次
  • 渐进复杂度
  • 体验最完整

缺点

  • 开发量最大
  • 界面稍复杂
